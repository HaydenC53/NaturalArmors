# NaturalArmors
Minecraft Spigot plugin that adds Stone and Wooden armors. May extend to other armors in the future.

--------------------------------------------------------------

## Wood armor has the following stats:
- Wood Helmet:
  - Armor: 0.5
  - Armor Toughness: 0
- Wood Chestplate:
  - Armor: 3.5
  - Armor Toughness: 0
- Wood Leggings:
  - Armor: 1.5
  - Armor Toughness: 0
- Wood Boots:
  - Armor: 1.5
  - Armor Toughness: 0

## Stone armor has the following stats:
- Stone Helmet:
  - Armor: 2.5
  - Armor Toughness: 0.5
- Stone Chestplate:
  - Armor: 5.5
  - Armor Toughness: 0.5
- Stone Leggings:
  - Armor: 4.5
  - Armor Toughness: 0.5
- Stone Boots:
  - Armor: 1.5
  - Armor Toughness: 0.5
